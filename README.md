# Library Management System (Java)

The Library Management System is a console-based Java application that helps manage books and members in a library. It allows users to add books, register members, issue and return books, search records, and store data permanently using text files.

---

## Features

* Add new books
* Register members
* Issue books to members
* Return issued books
* Search books by:

  * Title
  * Author
  * Category
* View all books
* View all members
* Sort books by:

  * Title
  * Author
* Automatic data saving and loading

---

## How to Run

1. Download or clone the repository
2. Open the project in any Java IDE (VS Code / IntelliJ / Eclipse)
3. Compile and run:

```
javac LibraryManager.java
java LibraryManager
```
